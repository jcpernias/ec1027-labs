Datos

Cargamos el paquete readxl para leer hojas de cálculo Excel y el paquete ggplot2 para realizar gráficos:

library(readxl)
library(ggplot2)

Leemos las variables del archivo vote1.xlsx:

vote1 <- read_xlsx("vote1.xlsx")

Regresión inicial

Usamos expendA para explicar el porcentaje de votos obtenidos por A: \[ voteA = \beta_0 + \beta_1 expendA + u \]

mod1 <- lm(voteA ~ expendA, data = vote1)

La estimación del parámetro \(\beta_1\) indica que por cada 1000$ adicionales de gasto el porcentaje de votos para el candidato A aumentará en 0.024 puntos porcentuales.

Calculamos los residuos y los valores ajustados de la variable dependiente y los almacenamos en la base de datos:

vote1$yhat <- fitted(mod1)
vote1$uhat <- resid(mod1)

Gastos del rival

Gráfico de los residuos con respecto a la variable explicativa:

qplot(x = expendA, y = uhat, data = vote1)

Gráfico de los residuos con respecto de los gastos del candidato B:

qplot(x = expendB, y = uhat, data = vote1)

Correlación entre expendA y expendB:

with(vote1,  cor(expendA, expendB))
